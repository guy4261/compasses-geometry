How to Use

🎯 Point Tool (P)

Click on canvas to place points

📏 Line Tool (L)

Click two points to connect them with a straight line

🧭 Compass Tool (C)

1. Click a point to set the center
2. Click a line to use its length as radius

👆 Select Tool (S)

• Click to select points
• Drag to move
• Arrow keys for precise movement
• Right-click or Delete/Backspace to remove

✨ Auto-Intersections

Intersection points are automatically created when circles intersect
